## 1.1 什么是 Ollama？

### 一句话介绍

Ollama 是一个**本地大模型运行工具**，让你在自己的电脑上运行 AI 模型，无需联网、无需 API 费用、数据完全私有。

### 为什么需要本地 AI？

想象一下这些场景：

| 场景 | 云端 API | 本地 Ollama |
|------|----------|-------------|
| **隐私敏感** | ❌ 数据上传到云端 | ✅ 数据完全本地 |
| **网络不稳定** | ❌ 无法使用 | ✅ 离线也能用 |
| **高频调用** | ❌ 费用爆炸 | ✅ 免费无限用 |
| **定制化需求** | ❌ 受限 | ✅ 完全可控 |
| **延迟要求** | ⚠️ 网络延迟 | ✅ 本地响应 |

### Ollama 的优势

1. **开箱即用** - 一条命令安装，一条命令运行
2. **模型丰富** - 支持 Llama、Qwen、DeepSeek 等主流模型
3. **跨平台** - Windows、macOS、Linux 全支持
4. **免费开源** - MIT 协议，完全免费
5. **社区活跃** - 持续更新，问题容易解决

### 适用人群

- ✅ **开发者** - 代码审查、Bug 调试、文档生成
- ✅ **内容创作者** - 文案写作、标题优化、脚本创作
- ✅ **学生** - 学习辅助、概念解释、论文润色
- ✅ **职场人士** - 周报生成、邮件润色、数据分析
- ✅ **隐私敏感用户** - 数据不出本地，完全可控

### 不适用场景

- ❌ 需要最新知识（本地模型知识有截止时间）
- ❌ 需要超大上下文（本地显存/内存有限）
- ❌ 需要多模态（纯文本模型，不支持图片/视频输入）

## 1.2 安装与配置

### 系统要求

| 配置项 | 最低要求 | 推荐配置 |
|--------|----------|----------|
| **操作系统** | Windows 10 / macOS 11 / Linux | 最新稳定版 |
| **内存** | 8 GB | 16 GB+ |
| **存储** | 10 GB 可用空间 | 50 GB+ SSD |
| **显卡** | 非必须 | NVIDIA 8GB+ 显存 |

> 💡 **提示：** 没有独立显卡也能用，只是推理速度慢一些。

---

### Windows 安装

#### 步骤 1：下载安装包

访问官网：https://ollama.com/

点击 **"Download for Windows"** 下载安装包。

#### 步骤 2：运行安装程序

双击下载的 `.exe` 文件，按提示安装。

![安装向导](./images/install-windows.png)

#### 步骤 3：验证安装

打开 PowerShell 或 CMD，运行：

```bash
ollama --version
```

看到版本号说明安装成功：

```
ollama version is 0.1.32
```

---

### macOS 安装

#### 方法 A：官网下载（推荐）

1. 访问 https://ollama.com/
2. 点击 **"Download for macOS"**
3. 双击 `.zip` 文件，拖拽到 Applications

#### 方法 B：Homebrew 安装

```bash
brew install ollama
```

#### 验证安装

```bash
ollama --version
```

---

### Linux 安装

#### 一键安装脚本

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

#### 手动安装

```bash
# 下载
curl -L https://ollama.com/download/ollama-linux-amd64.tgz -o ollama.tgz

# 解压
tar -C /usr -xzf ollama.tgz

# 启动服务
ollama serve
```

#### 设置为系统服务

```bash
sudo systemctl enable ollama
sudo systemctl start ollama
```

---

### 常见问题

#### Q1: 安装后找不到命令？

**解决：** 重启终端或添加环境变量。

Windows：
```powershell
$env:Path += ";C:\Program Files\Ollama"
```

#### Q2: 安装失败，提示权限不足？

**解决：** 用管理员身份运行安装程序。

#### Q3: 杀毒软件报毒？

**解决：** Ollama 是开源软件，可以添加到白名单。或者从官网重新下载。

## 1.3 第一个模型

### 选择你的第一个模型

对于新手，我推荐：

| 模型 | 大小 | 优点 | 缺点 | 适合场景 |
|------|------|------|------|----------|
| **Qwen2.5:7b** | 4.7 GB | 中文优秀，综合能力强 | 需要 8GB+ 内存 | 通用场景 ⭐ |
| **Llama3.1:8b** | 4.7 GB | 英文优秀，生态好 | 中文稍弱 | 英文内容 |
| **Phi-3:mini** | 2.3 GB | 轻量快速，低配置友好 | 能力稍弱 | 低配电脑 |
| **DeepSeek-Coder:7b** | 4.7 GB | 代码能力强 | 通用能力一般 | 编程辅助 |

> 💡 **建议：** 第一个模型选 **Qwen2.5:7b**，中文友好，综合能力强。

---

### 下载并运行模型

#### 步骤 1：拉取模型

打开终端，运行：

```bash
ollama pull qwen2.5:7b
```

等待下载完成（约 4.7 GB）：

```
pulling manifest
pulling 7b... 100%
pulling config... 100%
verifying sha256... 100%
success
```

> ⏱️ **时间提示：** 根据网速，通常需要 5-20 分钟。

#### 步骤 2：运行模型

```bash
ollama run qwen2.5:7b
```

看到 `>>>` 提示符说明运行成功。

#### 步骤 3：开始对话

输入你的问题：

```
>>> 你好，请用一句话介绍你自己
```

模型会回复：

```
你好！我是一个人工智能助手，可以帮你回答问题、写作、编程等。有什么我可以帮你的吗？
```

#### 步骤 4：退出

输入 `/bye` 或按 `Ctrl+D` 退出：

```
>>> /bye
```

### 常用命令速查

| 命令 | 说明 |
|------|------|
| `ollama pull <模型>` | 下载模型 |
| `ollama run <模型>` | 运行模型 |

| `ollama rm <模型>` | 删除模型 |
| `ollama cp <模型> <新名>` | 复制模型 |
| `ollama show <模型>` | 查看模型信息 |
