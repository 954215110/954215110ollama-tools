# 🦙 我做了一个 Ollama 增强工具

## 背景

最近在用 Ollama 跑本地 AI 模型（Qwen、Llama 等），发现官方的 Web UI 功能比较简单：
- 没有 Prompt 模板，每次都要重新输入
- 对话历史不好管理
- 想对比不同模型的回答，得打开多个标签页

所以我自己做了一个增强工具，开源了！

## 🔗 项目地址

https://github.com/954215110/954215110ollama-tools

## ✨ 功能特性

### 1. Prompt 模板库

内置 6 个常用模板：
- 代码助手
- 文案写作
- 翻译助手
- 学习导师
- 数据分析
- 润色改写

一键使用，不用每次都写"你是一个 XXX"了。

### 2. 对话历史管理

- 自动保存所有对话
- 支持查看最近 50 条记录
- 可以清空历史

### 3. 多模型对比

同时向多个模型提问，对比回答质量，方便选择最佳模型。

### 4. 简洁界面

轻量级 Web UI，无需复杂配置，启动即用。

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/954215110/954215110ollama-tools.git
cd ollama-tools

# 安装依赖
npm install

# 启动服务
npm start

# 访问 http://localhost:3000
```

## 💡 技术栈

- 后端：Node.js + Express
- 前端：原生 HTML/CSS/JS（无框架，轻量）
- AI：Ollama API
