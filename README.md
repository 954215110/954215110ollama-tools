## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 📝 **15+ Prompt 模板** | 代码/文案/翻译/周报/简历/小红书等，一键调用 |
| 💬 **对话历史** | 自动保存，支持搜索/导出 Markdown/PDF |
| 🔄 **多模型对比** | 同时问多个模型，对比回答质量 |
| 📤 **一键导出** | 对话内容导出为 Markdown 或 PDF |
| 🎨 **简洁界面** | 轻量级 Web UI，开箱即用 |

## 🚀 快速开始

### 前置要求

- [Ollama](https://ollama.ai) 已安装并运行
- Node.js 16+

### 安装

```bash
# 克隆项目
git clone https://github.com/yourusername/ollama-tools.git
cd ollama-tools

# 安装依赖
npm install

# 启动服务
npm start
```

### 使用

**本地访问：** `http://localhost:3000`

**外网访问（在线演示）：** `https://12wc0wo892531.vicp.fun`

## ❓ 为什么选择 Ollama Tools？

| 对比项 | Ollama 官方 UI | Ollama Tools |
|--------|---------------|--------------|
| Prompt 模板 | ❌ 无 | ✅ 15+ 内置模板 |
| 对话历史 | ⚠️ 基础 | ✅ 搜索 + 导出 |
| 多模型对比 | ❌ 无 | ✅ 支持 |
| 导出格式 | ❌ 无 | ✅ Markdown/PDF |
| 界面复杂度 | 中等 | 简洁易用 |

## 🛠️ 配置

编辑 `config.json` 自定义设置：

```json
{
  "port": 3000,
  "ollamaUrl": "http://localhost:11434",
  "defaultModel": "qwen2.5:7b"
}
```
