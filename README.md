## ✨ 功能特性

- 📝 **Prompt 模板库** - 内置常用 Prompt 模板，一键使用
- 💬 **对话历史管理** - 自动保存聊天记录，支持搜索/导出
- 🔄 **多模型对比** - 同时调用多个模型，对比回答质量
- 📤 **一键导出** - 导出对话为 Markdown/PDF
- 🎨 **简洁界面** - 轻量级 Web UI，无需复杂配置

---

## 🚀 快速开始

### 前置要求

- [Ollama](https://ollama.ai) 已安装并运行
- Node.js 16+

### 安装

```bash
# 克隆项目
git clone https://github.com/yourusername/ollama-tools.git
cd ollama-tools

# 安装依赖
npm install

# 启动服务
npm start
```

### 使用

打开浏览器访问：`http://localhost:3000`

## 📖 功能说明

### 1. Prompt 模板

内置常用模板：
- 代码助手
- 文案写作
- 翻译助手
- 数据分析
- 学习导师

### 2. 对话历史

- 自动保存所有对话
- 支持关键词搜索
- 可导出为 Markdown

### 3. 多模型对比

同时向多个模型提问，对比回答质量，方便选择最佳模型。

编辑 `config.json` 自定义设置：

```json
{
  "port": 3000,
  "ollamaUrl": "http://localhost:11434",
  "defaultModel": "qwen2.5:7b"
```
