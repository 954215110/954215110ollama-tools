## 正文

```
如题，自己折腾了一个 Ollama 的 Web UI，主要解决几个痛点：

1. 官方 UI 太简陋，没有 Prompt 模板
2. 聊天记录不好管理，想导出很麻烦
3. 想对比不同模型的回答质量

所以就做了这个工具，主要功能：

✅ 15+ 内置 Prompt 模板（代码/文案/翻译/周报/简历/小红书等）
✅ 对话历史自动保存，支持搜索和导出（Markdown/PDF）
✅ 多模型对比，同时问多个模型选最佳答案
✅ 界面简洁，开箱即用

项目地址：https://github.com/954215110/ollama-tools

在线演示：https://12wc0wo892531.vicp.fun

技术栈：
- Node.js + Express
- 原生 HTML/CSS/JS（没上框架，轻量）
- Ollama API

安装很简单：
```bash
git clone https://github.com/954215110/ollama-tools.git
cd ollama-tools
npm install
npm start
```

然后访问 http://localhost:3000 就行。

**前置要求：** 需要先安装 Ollama 并运行一个模型（比如 qwen2.5:7b）
