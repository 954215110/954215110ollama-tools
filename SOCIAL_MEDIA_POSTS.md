## 知乎（回答"有哪些好用的 AI 工具"类问题）

```
分享一个自己开源的项目：**Ollama Tools**

这是一个 Ollama 的增强版 Web UI，主要解决几个痛点：

1. 官方 UI 太简陋，没有 Prompt 模板
2. 聊天记录不好管理，想导出很麻烦
3. 想对比不同模型的回答质量

### 主要功能

- **15+ Prompt 模板**：代码助手、文案写作、翻译助手、周报生成、简历优化、小红书文案等
- **对话历史管理**：自动保存，支持搜索和导出（Markdown/PDF）
- **多模型对比**：同时向多个模型提问，对比回答质量
- **简洁界面**：轻量级 Web UI，开箱即用

### 安装使用

```bash
git clone https://github.com/954215110/ollama-tools.git
cd ollama-tools
npm install
npm start
```

访问 http://localhost:3000 即可。

### 项目地址

- GitHub: https://github.com/954215110/ollama-tools
- 在线演示：https://12wc0wo892531.vicp.fun

### 技术栈

- Node.js + Express
- 原生 HTML/CSS/JS
- Ollama API

完全免费开源，MIT 协议。如果觉得有用的话，欢迎点个 Star 支持！🙏

有任何问题或建议欢迎提 Issue～
```
