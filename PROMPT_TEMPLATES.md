## 🎯 详细模板

### 01 💻 代码助手

**模板：**
```
你是一个专业的程序员，请帮我解决以下编程问题：

【问题描述】
（在这里描述你的编程问题）

【编程语言】
（如：Python/JavaScript/Node.js 等）

【已尝试的方法】
（可选，描述你已经尝试过的解决方案）

【期望结果】
（你希望实现什么功能）
```

**使用示例：**
```
你是一个专业的程序员，请帮我解决以下编程问题：

【问题描述】
我想用 Python 读取 Excel 文件并生成图表

【编程语言】
Python

【已尝试的方法】
试过用 openpyxl，但不会画图表

【期望结果】
能读取 Excel 数据并生成柱状图
```

## 💡 使用技巧

### 1. 替换括号内容
将模板中 `（）` 里的内容替换成你的实际情况

### 2. 提供详细信息
信息越详细，AI 回答越精准

### 3. 多次迭代
如果第一次回答不满意，可以补充信息再问

### 4. 组合使用
可以组合多个模板，如"文案写作 + 小红书风格"

### 5. 保存常用
把自己常用的模板保存下来，快速调用
